# Excel Data Cleaning Project – U.S. Presidents Dataset

## Project Overview

This project focuses on cleaning a raw U.S. Presidents dataset using Microsoft Excel.

The dataset contained duplicate records, inconsistent text formatting, extra spaces, incorrect category values, salary values stored as text, and inconsistent date formats.

After cleaning, the dataset was standardized and prepared for analysis.

## Dataset Information

The dataset contains information such as:

- President Name
- Political Party
- Vice President
- Salary
- Date Updated
- Date Created

## Data Cleaning Tasks Performed

- Removed duplicate records
- Standardized President names using PROPER()
- Removed extra spaces using TRIM()
- Corrected inconsistent Party names
- Converted Salary from text format to numeric format
- Standardized Date formats
- Used Filters to identify data quality issues
- Applied Paste Special → Values to finalize cleaned data
